Cần kết nối Firebase để 2 máy giao tiếp

BƯỚC 1 — TẠO FIREBASE (MIỄN PHÍ)

1
Vào console.firebase.google.com → đăng nhập Google
2
Bấm "Add project" → đặt tên BTShare → Continue → Create
3
Trong menu trái → Build → Realtime Database
4
Bấm "Create database" → chọn Singapore → chọn "Start in test mode" → Enable
5
Copy link database ở trên cùng, dạng:
https://btshare-xxxxx-default-rtdb.asia-southeast1.firebasedatabase.app

BƯỚC 2 — DÁN LINK VÀO ĐÂY

⚡ Chỉ cần làm 1 lần. Sau đó gửi link app cho người kia, họ nhập cùng URL Firebase là 2 máy kết nối được nhau!
